# PPT Web Presentation

静态网页版本，适合直接发布到 GitHub Pages 或其他静态托管平台。

## 发布文件

- `index.html`：线上访问入口
- `preview.html`：本地编辑/预览副本
- `toonhub-ppt-theme-2chars-adidas-uploaded.html`：页面内嵌章节
- `assets/`：人物与页面资源

## GitHub Pages

1. 将本文件夹提交到 Git 仓库。
2. 在仓库 Settings -> Pages 中选择发布分支。
3. Source 选择仓库根目录。
4. 访问 Pages 生成的网址即可。

编辑后请先在页面里点“保存到文件”，再提交更新后的 `index.html` 和相关资源。
